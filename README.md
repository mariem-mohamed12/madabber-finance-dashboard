💰 Madabber Finance Dashboard

An Arabic financial dashboard UI designed to help users manage their money efficiently with a clean and modern interface.

📌 Overview

Madabber is a responsive dashboard that provides a complete overview of personal finances including balance, income, expenses, and recent transactions.

✨ Features

- Arabic RTL design 🇪🇬
- Fully responsive (mobile & desktop)
- Clean and modern UI
- Dashboard cards (balance, income, expenses)
- Transactions table
- Activity tracking section
- Progress bars and statistics
- Reusable utility classes (design system)

🛠️ Tech Stack

- HTML5
- CSS3 (Flexbox & Grid)
- Font Awesome
- Google Fonts (IBM Plex Sans Arabic)
